# 🎨 ArtMuse — Telegram бот по истории искусства

Интерактивный ИИ-гид, который исследует эпохи, рассказывает о художниках и проводит викторины.

## Запуск за 3 шага

**1. Установи зависимости**
```bash
pip install -r requirements.txt
```

**2. Создай файл `.env` и вставь ключи**
```bash
cp .env.example .env
# Открой .env и вставь свои ключи
```

- `TELEGRAM_TOKEN` — получи у [@BotFather](https://t.me/BotFather) командой `/newbot`
- `ANTHROPIC_API_KEY` — получи на [console.anthropic.com](https://console.anthropic.com)

**3. Запусти**
```bash
python main.py
```

## Что умеет бот

| Команда / действие | Результат |
|---|---|
| Написать год (напр. `1888`) | Эпоха, топ-5 художников, топ-3 шедевра |
| Написать имя художника | Биография, работы, анекдоты |
| Кнопка «Викторина» | 7 вопросов с разбором ошибок и счётом |
| `/menu` | Главное меню с выбором режима |

## Лимиты
- 20 сообщений в час на пользователя (можно изменить в `RATE_LIMIT_MAX`)
- История диалога: последние 10 сообщений на пользователя

## Деплой на сервер (опционально)
```bash
# Пример для Ubuntu
export TELEGRAM_TOKEN=...
export ANTHROPIC_API_KEY=...
nohup python main.py &
```
